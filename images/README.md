## 替换步骤

1. 将你的图片放入对应的文件夹
2. 确保文件名与上表一致
3. 刷新页面即可看到效果

## 注意事项

- 建议使用 PNG 格式，支持透明背景
- 图标类图片建议使用纯色背景或透明背景
- 横幅类图片建议使用高分辨率，保证显示效果
